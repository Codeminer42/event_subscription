# Codeminer 42 Event Subscription

- [x] Bootstrap
- [x] Sass
- [x] Bower
- [x] npm
- [x] Gulp
- [x] Autoprefixer
- [x] Browsersync

## Production URL
[eventos.codeminer42.com](http://eventos.codeminer42.com)

## Setup

```
$ npm install
```

## Run in your browser

```
$ gulp serve
```

## Build for production

```
$ gulp
```

## Deploy to gh-pages

```
$ gulp build
$ gulp deploy
```
